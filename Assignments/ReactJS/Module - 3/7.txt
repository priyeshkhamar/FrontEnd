Q - 7. How to install React Js on Windows, Linux Operating System? How to install NPM and How to check version of NPM?
--> 
- Steps to install React Js on Windows :
1. Install Node.js and NPM from Node.js official website. 
2. Verify Installation by using commands - 'node -v' & 'npm -v'.
3. Create a React Project by using commands - 'npx create-react-app project_name'.
4. Run the project by using commands - 'npm start'.


-Steps to install React Js on Linux :
1. Install Node.js and NPM by using commands - 'sudo apt update' & 'sudo apt install nodejs npm'.
2. Verify Installation by using commands - 'node -v & npm -v'.
3. Create New Project by using commands -  'npx create-react-app project_name'.
4. Run the project by using commands - 'npm start'.
